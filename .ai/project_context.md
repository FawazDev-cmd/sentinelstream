# SentinelStream Project Context

SentinelStream is a portfolio-first, real-time log intelligence platform. Its
planned purpose is to ingest structured server logs, detect deterministic
anomalies, group related failures into incidents, and produce evidence-based
human-readable explanations.

The project is in active development. Day 1 establishes only the Python and HTTP
application foundation. Ingestion, queues, persistence, detectors, incidents,
explanations, metrics, simulation, and dashboard behavior remain planned.

The core engineering values are explicit contracts, deterministic behavior,
framework-independent domain logic, typed boundaries, small milestones, and
documentation that describes only verified behavior as complete.
